## 🚀 Quick Deploy
1. Fork or clone this repo
2. Go to **Settings → Pages**
3. Source → **Deploy from a branch** → `main` → `/ (root)`
4. Your live site will be at `https://YOUR-USERNAME.github.io/syndicate-gateway`
